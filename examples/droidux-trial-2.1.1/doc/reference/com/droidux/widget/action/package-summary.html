package

com.droidux.widget.action

The widget.action.* package contains collection of widgets that can be used to present a set of actions (represented by ActionInterfaces.Item.ActionItem) to help user easily find information or execute certain functions.

NOTE: To avoid leaking the action window in your Activity (throwing the android.view.WindowLeaked exception), you need to dismiss the window after use, either by calling dismiss(), or by setting the setDismissOnClick(boolean) to true (the default is true). And to overcome this happening on configuration change (for example when the screen rotation is changed), you need to dismiss() the window before exiting the Activity, e.g. in onPause() or onDestroy()

See also this question at StackOverflow. 

public class ActionWindowTest extends Activity {
	private ActionWindowInterface mActWin;
	// ...
	public void showQuickAction(View anchor) {
		mActWin = new QuickActionBar(this);
		mActWin.beginLayout()
				.addAction(action1)
				.addAction(action2)
				.addAction(action3)
				.endLayout();

		mActWin.show(anchor);
		//....
	}


	@Override
	public void onPause() {
		super.onPause();

		// dismiss the action window
		if(mActWin != null) {
			mActWin.dismiss();
			mActWin=null;
		}
	}
}
