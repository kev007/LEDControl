public interface

NoteInterface

com.droidux.widget.editor.NoteInterface
Known Indirect Subclasses
AutoCompleteNoteEditor, MultiAutoCompleteNoteEditor, NoteEditor, NoteView
AutoCompleteNoteEditor A subclass of AutoCompleteTextView with additional features, such as options to draw lines, color margins, gradient color background, etc to simulate notepad paper. 
MultiAutoCompleteNoteEditor A subclass of MultiAutoCompleteTextView with additional features, such as options to draw lines, color margins, gradient color background, etc to simulate notepad paper. 
NoteEditor A subclass of EditText with additional features, such as options to draw lines, color margins, gradient color background, etc to simulate notepad paper. 
NoteView A subclass of TextView with additional features, such as options to draw lines, color margins, gradient color background, etc to simulate notepad paper. 

Class Overview

Represents contracts that need to be implemented by notes widgets.

XML Attributes

dux_drawLine

Since: API Level

Controls whether a horizontal line should be drawn across the note widget. Default is true.

Must be a boolean value, either "true" or "false".

This may also be a reference to a resource (in the form "@[package:]type:name") or theme attribute (in the form "?[package:][type:]name") containing a value of this type.

Related Methods
  • setShouldDrawLine(boolean)

dux_drawMargin

Since: API Level

Controls whether to draw a color margin on the left side of the widget. Default is false.

Must be a boolean value, either "true" or "false".

This may also be a reference to a resource (in the form "@[package:]type:name") or theme attribute (in the form "?[package:][type:]name") containing a value of this type.

Related Methods
  • setShouldDrawMargin(boolean)

dux_lineColor

Since: API Level

Specifies the line color to be used if the dux_drawLine is set to true. If the line color is not specified and the dux_drawLine is true, a color will be automatically calculated.

May be a reference to another resource, in the form "@[+][package:]type:name" or to a theme attribute in the form "?[package:][type:]name".

May be a color value, in the form of "#rgb", "#argb", "#rrggbb", or "#aarrggbb".

Related Methods
  • setLineColor(int)

dux_marginColor

Since: API Level

Specifies the margin color to be used if the dux_drawMargin is set to true.

May be a reference to another resource, in the form "@[+][package:]type:name" or to a theme attribute in the form "?[package:][type:]name".

May be a color value, in the form of "#rgb", "#argb", "#rrggbb", or "#aarrggbb".

Related Methods
  • setMarginColor(int)

dux_marginType

Since: API Level

The type of the paper color for use as background of text editing widgets.

Must be one of the following constant values.

ConstantValueDescription
solid0 Use solid color.
gradient1 Use gradient color. The gradient color will be automatically created based on the color specified by the dux_paperColor attribute.

Related Methods
  • setMarginType(int)

dux_marginWidth

Since: API Level

Specifies the width of the margin, if the dux_drawMargin is set to true. Its value should be a dimension (such as "5dip").

Must be a dimension value, which is a floating point number appended with a unit such as "2.5sp". Available units are: px (pixels), dp (density-independent pixels), sp (scaled pixels based on preferred font size), in (inches), mm (millimeters).

Related Methods
  • setMarginWidth(int)

dux_paperColor

Since: API Level

Specifies the paper color for use as background of text editing widgets. If the widget already has a background, the color specified by this attribute will be drawn over the background, so you might need to adjust the opacity of the color to still see the background. You can use a transparent color to not draw a color on the background.

May be a reference to another resource, in the form "@[+][package:]type:name" or to a theme attribute in the form "?[package:][type:]name".

May be a color value, in the form of "#rgb", "#argb", "#rrggbb", or "#aarrggbb".

Related Methods
  • setPaperColor(int)
